* Init project
    * npm init
    * npm add -D typescript nodemon ts-node @types/express @types/node
    * npm add express pg typeorm dotenv reflect-metadata
    * npm add express-async-errors

* Rodar projeto
    * npm run dev

* migrations
    * npm run migration:generate
    * npm run migration:run

    * https://typeorm.io/
